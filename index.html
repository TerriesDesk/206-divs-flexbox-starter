HTML
Short for HyperText Markup Language, the authoring language used to create documents on the World Wide Web. HTML defines the structure and layout of a Web document by using a variety of tags and attributes.

CSS
Short for Cascading Style Sheets, CSS gives both Web site developers and users more control over how pages are displayed. With CSS, designers can create style sheets that define how different elements appear. These style sheets can then be applied to any Web page.

JavaScript
JavaScript is a scripting language first developed by Netscape to enable Web authors to design interactive sites. Javascript can interact with HTMLsource code, enabling Web authors to spice up their sites with dynamic content.
